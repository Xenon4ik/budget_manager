# Budget Manager

## Описание проекта

**Budget Manager API** — это REST-сервис для управления личным бюджетом, разработанный на FastAPI.

Приложение позволяет пользователю:

- регистрироваться и входить в систему;
- создавать категории доходов и расходов;
- добавлять финансовые операции;
- создавать цели накоплений;
- анализировать расходы;
- получать простой финансовый план для достижения цели накопления.

Проект разработан в рамках учебного задания по созданию backend-сервиса с использованием FastAPI, реляционной базы данных, авторизации, CRUD-операций, бизнес-логики и автоматических тестов.

---

## Цель проекта

Цель проекта — разработать полноценный REST API для учета личных финансов.

Сервис помогает пользователю понять:

- сколько денег он тратит по категориям;
- сколько может откладывать каждый месяц;
- достижима ли его финансовая цель;
- какие категории расходов можно сократить.

---

## Основная функциональность

В проекте реализованы:

- CRUD-операции для категорий;
- CRUD-операции для транзакций;
- CRUD-операции для целей накоплений;
- регистрация пользователей;
- вход пользователей в систему;
- JWT-аутентификация;
- защищенные эндпоинты;
- бизнес-эндпоинт для расчета плана накоплений;
- автоматические тесты на основе `TestClient`;
- подключение к реляционной базе данных SQLite.

---

## Предметная область

Проект относится к предметной области **личных финансов**.

Пользователь может вести учет доходов и расходов, распределять операции по категориям и ставить финансовые цели, например:

- накопить на отпуск;
- собрать финансовую подушку;
- накопить на покупку техники;
- снизить расходы на развлечения или кафе.
